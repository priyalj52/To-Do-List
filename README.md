# To-Do-List
A simple Todo-App built using HTML,CSS,JAVASCRIPT ,NODEJS,EJS,MONGOOSE.
USER CAN WRITE HIS/HER TODO'S AND DELETE IT BY CLICKING ON DELETE BUTTON
